### Descrição narrativa
* Receber um valor
* Calcular o valor vezes dois
* Verificar se o valor é maior ou igual a 30 e menor ou igual a 100

### Pseudo-código
```
programa {
  funcao inicio() {
    inteiro numb, res

    escreva("Escreva um número inteiro qualquer: ")
    leia(numb)
    res = numb*2

    se(res>=30 e res<=100){
      escreva("O número está entre 30 e 100 e ele é: ", res)
    }
  }
}
```
