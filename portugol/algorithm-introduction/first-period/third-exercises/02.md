## Descrição 
  - Receber os valores
  - Validar se a é maior que b ou b é maior que a
  - Escrever validação

## Fluxograma 
## Pseudo-código 

```
programa {
  funcao inicio() {
    inteiro a, b, dif;
    leia(a)
    leia(b)

    se(a > b){
      dif = a-b
      escreva(dif)
    }senao{
      dif = b-a
      escreva(dif)
    }
  }
}
```
