## Descrição
  - Receber valores
  - Calcular soma dos valores
  - Validar se a soma dos valores são maior que 100
  - Retornar validação

## Fluxograma

## Pseudo-código

```
programa {
  funcao inicio() {
    inteiro a, b, c, sum

    leia(a)
    leia(b)
    leia(c)

    sum = a+b+c

    se(sum>100){
      escreva(sum)
    }senao{
      escreva("Não é maior que 100")
    }
  }
}
```
