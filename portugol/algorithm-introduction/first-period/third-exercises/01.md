## Descrição
  - Receber a idade
  - Validar se a idade é maior que 18 ou menor
  - Retornar validação

## Fluxograma

## Pseudo-código

```
programa {
  funcao inicio() {
    inteiro idade;

    escreva("Digite a idade do aluno: ")
    leia(idade)

    se(idade < 18){
      escreva("Menor de idade!")
    }senao{
      escreva("Maior de idade!")
    }
  }
}
```
